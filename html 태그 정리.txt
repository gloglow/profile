html 태그 정리

<strong> -> 볼드체
<u> -> 밑줄
<h1> -> 제목

<br> -> 줄바꿈
: 닫지않아도 된다

<p> -> 단락
: 여러 줄 바꿈이 안되지만 css로 보완가능
**CSS**
<p style="margin-top:40px;">
단락 간 여백 설정이 가능하다.

<img @@@@> -> 이미지넣기
** 속성 문법 ** => @@@@ 부분에다가 넣어줌. 순서 상관 x
src="###.jpg"
width="###"

<li> -> 리스트 만들기
<ul> -> 리스트 구분시켜주는 부모태그.
<ol> -> 리스트 구분시켜주는 부모태그2. 앞에 숫자 자동으로 넘버링

<title> -> 웹페이지의 제목.

<head> -> 본문이 아닌 
<body> -> 본문에.
<html> -> 헤드와 바디를 한번에 묶는 최상위 태그.
<!doctype html> -> 가장 윗줄에 써주는 관용어

<meta charset="utf-8"> -> 글씨깨짐 방지. utf-8 방식으로 브라우저가 열도록.

<a href="주소"> -> 하이퍼링크
target="_blank" 를 붙여주면 새 탭이 열림.
title="~~~" 를 붙여주면 링크에 마우스 올려놨을 때 툴팁이 열림.
